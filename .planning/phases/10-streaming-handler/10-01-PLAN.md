Create `src/python/streaming.rs` with `StreamingHandler` struct and `invoke_streaming` method.

**StreamingHandler responsibilities:**
- Wraps a Python async generator (coroutine) via PythonAsyncFuture
- Polls the generator repeatedly inside a loop (not a single await — persistent loop)
- Each yield from the generator is treated as a message to process
- StopAsyncIteration exception signals end of stream (graceful stop)
- Other exceptions propagate as ExecutionResult::Error

**invoke_streaming method signature:**
```rust
pub async fn invoke_streaming(
    &self,
    ctx: &ExecutionContext,
    initial_message: OwnedMessage,
) -> ExecutionResult
```

**Implementation structure:**

```rust
use crate::python::async_bridge::PythonAsyncFuture;
use crate::python::context::ExecutionContext;
use crate::python::execution_result::ExecutionResult;
use std::sync::Arc;

// StreamingHandler wraps a streaming async handler invocation
pub struct StreamingHandler {
    callback: Arc<Py<PyAny>>,
    middleware: Option<Vec<Arc<Py<PyAny>>>>,
}

impl StreamingHandler {
    pub fn new(callback: Arc<Py<PyAny>>, middleware: Option<Vec<Arc<Py<Any>>>>) -> Self {
        Self { callback, middleware }
    }

    /// Drives a Python async generator until StopAsyncIteration or error.
    /// Called repeatedly by the streaming worker loop.
    pub async fn invoke_streaming(&self, ctx: &ExecutionContext) -> ExecutionResult {
        // Step 1: Create coroutine object via Python GIL
        let coro: Py<PyAny> = Python::attach(|py| {
            // Build message dict for the initial message
            let py_msg = message_to_pydict(py, &initial_message, None);
            // Call the async generator function — returns coroutine object
            self.callback.call1(py, (py_msg,)).expect("must be async generator")
        });

        // Step 2: Wrap in PythonAsyncFuture and poll in a loop
        let mut future = PythonAsyncFuture::from(coro);

        loop {
            match future.await {
                ExecutionResult::Ok => {
                    // Generator exhausted normally
                    break;
                }
                ExecutionResult::Error { reason, exception, traceback } => {
                    // Error during yield — propagate
                    return ExecutionResult::Error { reason, exception, traceback };
                }
                // Handle other result types (Rejected, Timeout) — treat as error
                _ => {
                    return ExecutionResult::Error {
                        reason: FailureReason::Terminal(TerminalKind::HandlerPanic),
                        exception: "StreamingError".to_string(),
                        traceback: "Unexpected result in streaming loop".to_string(),
                    };
                }
            }
        }

        ExecutionResult::Ok
    }
}
```
