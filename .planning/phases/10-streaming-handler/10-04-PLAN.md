2. **Add pause_topic/partition methods to QueueManager:**
   ```rust
   /// Pauses Kafka consumption for a specific topic/partition.
   /// Used for backpressure when streaming handler queue is full.
   pub(crate) fn pause_partition(&self, topic: &str, partition: i32) {
       let mut state = self.handler_states.lock();
       if let Some(state) = state.get_mut(topic) {
           state.paused_partitions.insert(partition);
           tracing::debug!(topic, partition, "pausing partition due to backpressure");
       }
   }

   /// Resumes Kafka consumption for a specific topic/partition.
   pub(crate) fn resume_partition(&self, topic: &str, partition: i32) {
       let mut state = self.handler_states.lock();
       if let Some(state) = state.get_mut(topic) {
           state.paused_partitions.remove(&partition);
           tracing::debug!(topic, partition, "resuming partition after backpressure relief");
       }
   }
   ```

3. **HandlerState struct update** (if paused_partitions field doesn't exist):
   ```rust
   struct HandlerState {
       // ... existing fields
       paused_partitions: HashSet<i32>,
   }
   ```

4. **Streaming handler registration with dedicated capacity:**
   Find where streaming handlers register. Add:
   ```rust
   let capacity = STREAMING_BUFFER_CAPACITY;
   let rx = self.register_handler_with_semaphore(topic, capacity, None);
   ```

5. **Backpressure callback wiring:**
   When `try_send` fails with Full and policy returns `BackpressureAction::PausePartition`:
   - Call `queue_manager.pause_partition(topic, partition)`
   - Emit metric: `kafpy_streaming_backpressure_pause_total{topic, partition}++`

When queue depth drops below threshold (resume):
- Call `queue_manager.resume_partition(topic, partition)`
- Emit metric: `kafpy_streaming_backpressure_resume_total{topic, partition}++`

**Important:** The actual Kafka consumer pause/resume calls happen in the dispatcher or consumer loop — this is where we store the paused state. The consumer loop checks `is_paused(topic, partition)` before poll.
