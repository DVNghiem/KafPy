   - `fn record_counter(&self, name: &str, labels: &[(&str, &str)])`
   - `fn record_histogram(&self, name: &str, value: f64, labels: &[(&str, &str)])`

   - Wraps `Vec<(&'static str, String)>`

   - Lexicographic sort on key prevents cardinality explosion (Pitfall 5)

   - `record_batch_size(labels: &MetricLabels, size: usize)` — emits kafpy.handler.batch_size histogram

4. **QueueSnapshot struct** (OBS-06):
   - `queue_depth: std::sync::atomic::AtomicUsize`
   - `inflight: std::sync::atomic::AtomicUsize`
