# Principle of Least Exposure (D-02)

Items that MUST be pub (Python-accessible):
- Types used in #[pyclass] (Consumer, KafkaMessage, ConsumerConfig, ProducerConfig, PyProducer)
- Functions used in #[pyfunction] (get_runtime_snapshot, register_status_callback)
- Re-exports needed for the pymodule to find them

Items that should be pub(crate):
- Types used across multiple internal modules (e.g., ConsumerRunner used by pyconsumer and coordinator)
- Traits used internally (Executor, DlqRouter, etc.)

Items that should be private (default):
- Implementation details not needed outside their module
- Internal helpers and utilities
