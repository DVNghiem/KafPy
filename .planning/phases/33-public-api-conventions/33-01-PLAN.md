# PKG-01 Required Surface (per REQUIREMENTS.md)

`import kafpy` must expose:
- KafPy (or ConsumerApp)
- ConsumerConfig
- RoutingConfig
- RetryConfig
- BatchConfig
- ConcurrencyConfig
- KafkaMessage
- HandlerContext
- HandlerResult
- KafPyError
- ConsumerError
- HandlerError
- ConfigurationError

# D-03: Full surface re-export

All public types from submodules must be re-exported at the package level for discoverability.

# D-06: No underscore-prefixed or raw Rust names

- No `_private` or `_internal` names in __all__
- No `PyConsumer`, `PyProducer`, `PyError` (raw Rust struct names) exposed to Python
- Python-facing names use PascalCase for classes
